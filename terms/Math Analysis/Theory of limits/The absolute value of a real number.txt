Абсолютное значение реального числа — это неотрицательное значение числа без учёта знака. Абсолютное значение числа всегда либо положительное, либо равно 0. Абсолютное значение обозначается как |x| или abs(x). Например, числа 2 и -2 имеют расстояние 2 от 0, поэтому их абсолютное значение одинаково, несмотря на разный знак: |2| = 2, |-2| = 2. Абсолютное значение 0 равно 0.
The absolute value of a real number is the non-negative value of the number without considering the sign. The absolute value of a number is always either positive or equal to 0. The absolute value is denoted as |x| or abs(x). For example, the numbers 2 and -2 have a distance of 2 from 0, so their absolute value is the same despite their different sign: |2| = 2, |-2| = 2. The absolute value of 0 is 0.
