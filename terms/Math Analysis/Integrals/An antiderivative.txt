Первообразная функция — это функция F(x), производная которой равна данной функции f(x), то есть F'(x) = f(x). Первообразные функции определяются с точностью до постоянной, так как производные констант равны нулю. Например, если f(x) = 2x, то одной из первообразных будет F(x) = x² + C, где C — произвольная константа. Первообразная используется для вычисления интегралов и позволяет находить площади под графиками функций.
An antiderivative is a function F(x) whose derivative equals a given function f(x), that is, F'(x) = f(x). Antiderivatives are defined up to a constant since the derivative of a constant is zero. For example, if f(x) = 2x, one antiderivative would be F(x) = x² + C, where C is an arbitrary constant. Antiderivatives are used to compute integrals and allow for finding areas under the graphs of functions.
